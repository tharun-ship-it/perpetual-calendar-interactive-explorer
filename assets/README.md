# Assets Directory

This directory contains static assets for the project.

## Files

- `screenshot.png` - Application screenshot for README

## Adding a Screenshot

To add a screenshot:

1. Run the application:
   ```bash
   python src/calendar_app.py
   ```

2. Take a screenshot of the full window

3. Save it as `screenshot.png` in this directory

4. Recommended dimensions: 1200x800 pixels
